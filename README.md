# Cyborg MMO7 Revival

Cyborg MMO7 Revival is a community revival of the original Cyborg MMO7 addon for World of Warcraft. The addon is a companion for the Cyborg / Mad Catz M.M.O.7 and M.M.O.7+ mice.

The original addon was abandoned after WoW patch 6.1.0 and gradually broke as the WoW addon API changed. This revival cleans up and updates the addon for modern World of Warcraft, with support for Retail, Classic, and Era.

You can find the project on CurseForge here:

https://www.curseforge.com/wow/addons/cyborg-mmo7-revival

The original abandoned addon is available here:

https://www.curseforge.com/wow/addons/cyborg-mmo7

## Profiles

Profiles for both the original M.M.O.7 and the newer M.M.O.7+ are included in the [CurrentProfile](CurrentProfile) folder:

- `Mmo7Wow.pr0` for the original M.M.O.7
- `MMO7+ WoW.profile` for the M.M.O.7+

The M.M.O.7+ was released after this revival work began. The addon works with it, though more M.M.O.7+-specific improvements may be added over time.

## Known Issues

- M.M.O.7+ does not support mode switching and is currently limited to a single mode, the red mode.
- Toy Box items do not work yet.
- Transmog profiles have not been implemented yet.
